for문 사용하기

  • v-for="item in 배열이름"
  • :key는 unique한 값으로 입력
  • cars : ['hyundai','kia','bmw','volvo','benz','kia']
  • fruits : ['strawberry', 'apple', 'banana','mango','blueberry']
  • students : [ { name : 'Kim', age : 18 }, { name : 'Oh', age : 19 }, { name : 'Park', age : 20 } ]

기초

        <ul>
            <li v-for="item in cars" :key="item">
                { { item } }
            </li>
        </ul>
  • {{ item }}

배열 index를 사용할 경우

        <ul>
            <li v-for="(item,i) in fruits" :key="i">
                { { item } }
            </li>
        </ul>
  • {{ item }}

문자열이 json list일 경우

        <ul>
            <li v-for="(item,i) in students" :key="i">
                { { item.name } } ({ { item.age } })
            </li>
        </ul>
  • {{ item.name }} ({{ item.age }})
